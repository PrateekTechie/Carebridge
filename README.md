# CareBridge AI

Bridging the gap between hospital and home recovery.

## 🚀 Overview
CareBridge AI is a HealthTech platform designed to solve the critical care gap after hospital discharge. It enables real-time patient monitoring, AI-powered alerts for doctors, and structured recovery tracking.

## 💡 Problem
Patients often face confusion, medication errors, and lack of monitoring after leaving the hospital, leading to high readmission rates and caregiver stress.

## ✅ Solution
CareBridge AI provides:
- Real-time patient monitoring
- AI-driven alerts for doctors
- Recovery tracking dashboard
- Medication and symptom tracking

## 🛠 Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
